### В Telegram:

**Проблема:** GitHub Pages не может обратиться на localhost!

**Решение 1: Orchestrator с публичным URL**

1. Запустите ngrok/loca.lt:
   ```bash
   lt --port 8000
   ```

2. Скопируйте URL (например: `https://abc123.loca.lt`)

3. Обновите Web App URL в @BotFather:
   ```
   https://xctpejiokx.github.io/recruit_bot/index.html
   ```

4. В `app.js` замените API_URL на публичный:
   ```javascript
   const API_URL = 'https://abc123.loca.lt/vacancies';
   ```

**Решение 2: Render.com (24/7)**

1. Зарегистрируйтесь на https://render.com
2. Подключите репозиторий: `xCTPEJIOKx/recruit_bot`
3. Добавьте переменные окружения из `.env`
4. Deploy!

Получите URL: `https://recruit-orchestrator.onrender.com`

## ❗ ТИПИЧНЫЕ ПРОБЛЕМЫ

### Web App пишет "Ошибка загрузки вакансий"

**Причина:** API URL не совпадает с сервером

**Решение:**
- Локально: `http://localhost:8000/static/index.html`
- API: `http://localhost:8000/vacancies`

### Telegram не отправляет уведомления

**Причина:** Неверный chat_id

**Решение:**
```bash
python scripts/get_chat_id.py
```
Обновите `TELEGRAM_ADMIN_CHAT_ID` в `.env`

### Localtunnel не работает

**Причина:** Сессия истекла

**Решение:**
```bash
pkill -f "lt --port"
lt --port 8000
```
Скопируйте новый URL и обновите `app.js`
