# 📢 Настройка Classifieds Agent для реальной публикации

## ✅ Рабочие площадки (бесплатно)

### 1. HH.ru (HeadHunter)

**Регистрация:**
1. Перейдите на https://hh.ru/employer
2. Нажмите "Зарегистрироваться"
3. Выберите "Работодатель"
4. Заполните данные компании

**Публикация вакансий:**
- Бесплатно: 1 вакансия в месяц (для ИП и ООО)
- Можно публиковать вручную через личный кабинет

**API (опционально):**
- Получите ключи на https://api.hh.ru
- Добавьте в `.env`:
```bash
HH_CLIENT_ID=your_id
HH_CLIENT_SECRET=your_secret
```

## 🔧 Настройка Classifieds Agent

### Вариант 1: Ручная публикация (бесплатно)

1. Зарегистрируйтесь на площадках выше
2. Публикуйте вакансии вручную через личный кабинет
3. В `.env` укажите контакты для связи

### Вариант 2: Автоматизация через Selenium

Для автоматической публикации нужен:
1. Зарегистрированные аккаунты
2. Логины/пароли в `.env`:
```bash
HH_LOGIN=your_email
HH_PASSWORD=your_password
RABOTA_LOGIN=your_email
RABOTA_PASSWORD=your_password
```
